# BJJInsights | Blog de Jiu-Jitsu

![bjjinsights](https://github.com/MarceloMafraBJJ/bjjinsights/assets/84472778/9f09144f-6acb-4031-a76d-523c68d85bd1)

Features:

- Nextjs
- TailwindCSS
- NextAuth
- Prisma
- MongoDB

### Install packages

```shell
npm i
```

### Setup .env file

```js
GOOGLE_ID=
GOOGLE_SECRET=
NEXT_PUBLIC_API_URL=
NEXTAUTH_URL=
NEXTAUTH_SECRET=
FIREBASE_API_KEY=
DATABASE_URL=
NODEMAILER_EMAIL=
NODEMAILER_PW=
```

### Run

```shell
npm run dev
```
